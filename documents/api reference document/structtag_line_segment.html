unsigned char* tagLineSegment::linesConfidenceCoefficients

The confidence coefficients for lines. There are 4 coefficients in this set: linesConfidenceCoefficients[0] is average positive amplitude; linesConfidenceCoefficients[1] is max positive amplitude; linesConfidenceCoefficients[2] is average negative amplitude; linesConfidenceCoefficients[3] is max negative amplitude.
