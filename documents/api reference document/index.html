Dynamsoft Barcode Reader - API Reference Documentation

Introduction

Dynamsoft's Barcode Reader SDK enables you to efficiently embed barcode reading functionality in your application using just a few lines of code. This can save you from months of added development time and extra costs. With the Barcode Reader SDK, you can decode barcodes from various image file formats (BMP, JPEG, PNG, TIFF, and PDF) as well as device-independent bitmap (DIB) obtained from cameras and scanners, etc. The SDK comes with multiple editions which include Windows, JavaScript, Mobile, Linux, Raspberry Pi, and Mac.
This is the API reference for the Windows and Linux Edition (C/C++, .NET). If you are new to the product, you may refer to the Developer Guide for Windows or Developer Guide for Linux to get started. You can also find a copy of the Developer Guide in the installation folder.
